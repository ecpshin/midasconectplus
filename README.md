# Softbase

É o código base para do sistema da Midas Conecta+, desenvolvido para auxiliar nas tarefas da empresa

## Módulos
- CRUD de Propostas
    - Edição de proposta poderá ser realizado via formulário ou modal
    - CRUD de Comissões integrada as propostas
- CRUD de Clientes
- CRUD de Tabela Auxiliares
